# Projeto Microblog com Flask

Este projeto é um microblog desenvolvido com o framework Flask. O objetivo é fornecer uma plataforma simples para os usuários postarem atualizações e interagirem uns com os outros.

## Funcionalidades
- **Cadastro de Usuários:** Os usuários podem criar contas e fazer login para acessar suas funcionalidades.
- **Postagens:** Os usuários podem criar novas postagens, visualizar postagens existentes e interagir com elas.
- **Comentários:** Os usuários podem comentar nas postagens de outros usuários.
- **Interface Limpa:** O microblog oferece uma interface de usuário intuitiva e responsiva.

## Estrutura do Projeto
- **app/**: Contém todos os arquivos principais do aplicativo Flask.
- **templates/**: Arquivos HTML do frontend.
- **static/**: Arquivos CSS e JavaScript.
- **requirements.txt**: Lista de dependências necessárias para executar o projeto.

## Como Executar o Projeto
1. Clone o repositório: `git clone <URL_DO_REPOSITORIO>`
2. Navegue até o diretório do projeto: `cd microblog`
3. Instale as dependências: `pip install -r requirements.txt`
4. Execute o aplicativo: `flask run`
5. Acesse em `http://127.0.0.1:5000`
